Change the App Id and App Name

Every Android/iOS Application on Google Play or on AppStore has a unique id (Package name for android and Bundle Identifier on iOS) , for this application too we need to use a unique id , for that you need to choose a unique id with three part for example : com.companyname.appname , now to change you package name follow the video bello :

Please note

  • The only special characters allowed on package name its underscore
  • Do not use _ or any other special characters for iOS id
  • Google Play/AppStore will not allow you to change the package name after published
Commands used in video 
flutter pub get
flutter pub run flutter_application_id:main -f application_id.yaml

Facebook Configuration

After config Firebase now you need to config the facebook Application (this configuratio required to use facebook login + Facebook Network Audiance )

use the commande bellow to generate the hashkey :

For MacOS

keytool -exportcert -alias androiddebugkey -keystore ~/.android/debug.keystore | openssl sha1 -binary | openssl base64

For Windows

keytool -exportcert -alias androiddebugkey -keystore %HOMEPATH%\.android\debug.keystore | openssl sha1 -binary | openssl

Link The Android App With Admin panel

After the android app configuration and the admin panel configuration now you need to link it .
to do this go to : 

Application>lib>api>api_config.dart
and add your api url there (if your admin panel its http://admin.example.com you api url will be http://admin.example.com/api/) 
static const String api_url = "https://football.virmana.com/api/";

Then add your item purchase code in 
  static const String item_purchase_code  = "xxxxxx-xxxxxx-xxxxxxx-xxxxxxx-xxxxxxx";

Generate Release APK/AppBundle

Commands used to create keystore

For MacOS
 keytool -genkey -v -keystore keystore.jks -keyalg RSA -keysize 2048 -validity 10000 -alias key
For Windows
keytool -genkey -v -keystore c:\Users\USER_NAME\keystore.jks -storetype JKS -keyalg RSA -keysize 2048 -validity 10000 -alias key

Commands used to generate apk or appbundle file

For APK
flutter build apk
For appbundle
flutter build appbundle
